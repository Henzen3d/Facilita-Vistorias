       {

         imovel: { endereco, numero, complemento, bairro, cidade, estado },
         vistoriador: { id, nome },
         ambientes: [{ id, nome, ordem, items: [{ id, nome, status, descricao, descricaoFinal, descricaoEditada, estadoConservacao, analisadoEm, midias: [{id,tipo,url}], analise: { provedor, analise, usedFallback, createdAt } | null }] }],

       }

    1. Require session + same authz as above (item must belong to vistoria id in path).
    2. Body Zod: { descricaoFinal: string min 1, estadoConservacao?: "NOVO"|"BOM"|"REGULAR"|"RUIM", approve?: boolean }.

       Simpler rule: descricaoEditada = descricaoFinal.trim() !== (item.descricao ?? "").trim(); if approve true and texts equal, descricaoEditada false.
    5. Update item: descricaoFinal, estadoConservacao if provided, descricaoEditada, status = REVISADO.
    6. Return updated item JSON.

    Do not generate PDF. Do not change Vistoria.status to FINALIZADA.
  </action>
