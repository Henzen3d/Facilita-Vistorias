```ts
export const DescriptionOutputSchema = z.object({
  itemState: z.enum(["Novo", "Bom", "Regular", "Ruim"]),
  technicalDescription: z.string(),
});
export type DescriptionOutput = z.infer<typeof DescriptionOutputSchema>;

class AIRouter {
  async transcribeAudio(audioBuffer: Buffer, mimeType: string): Promise<string>;
  async generateDescription(
    photoBuffer: Buffer,
    mimeType: string,
    transcript: string
  ): Promise<DescriptionOutput>;
}
```

Model config (AI-SPEC §4):
- Primary: process.env.GEMINI_MODEL || "gemini-1.5-flash" (env.example also lists gemini-2.0-flash — use GEMINI_MODEL env with fallback "gemini-2.0-flash" to match env.example, documented in code comment)
- Fallback STT: whisper-1 (OPENAI_AUDIO_MODEL)
- Fallback vision: OPENAI_VISION_MODEL || "gpt-4o-mini"
- Temperature: 0.1
- Max tokens: 300 for description
- Failover on 429, 503, or timeout > 10s

Banned terms (never in output): "laudo", "laudo técnico", "perícia" (and common variants with accents: perícia/pericia)

Installed SDKs only: openai, @google/generative-ai, @anthropic-ai/sdk, zod — do NOT add LangChain/LangGraph/CrewAI/Vercel AI SDK.
