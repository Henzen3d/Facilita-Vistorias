## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Unauthenticated internet → public report | Token is sole credential for client access |
| Finalize API → PDF worker | Trusted staff triggers document generation |
| PDF worker → storage | PDF bytes stored for download/retention |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-18 | Spoofing | Public token URL | mitigate | Cryptographically random UUID tokenPublico; expiry via expiracaoToken; invalid/expired → not found |
| T-03-19 | Information Disclosure | Public report | mitigate | Expose only report fields; no CPF full display requirement in Phase 3 — omit documento or mask if present |
| T-03-20 | Tampering | Finalize before review | mitigate | Server-side check all items REVISADO/FINALIZADO before enqueue |
| T-03-21 | Elevation of Privilege | Finalize API | mitigate | Session + empresa/assignment checks |
| T-03-22 | Denial of Service | Puppeteer | mitigate | BullMQ retries; single concurrency for PDF worker recommended (concurrency: 1) |
| T-03-23 | Information Disclosure | QR/PDF | accept | Public URL is intentional share surface; expiry limits window |
| T-03-SC | Tampering | npm installs | accept | puppeteer/qrcode/jsonwebtoken already in package.json — no new packages |
</threat_model>
